## Plurals
### Masculine
+ и (Долар Долари)
+ і (Якщо -ь) (Готель Готелі)
### Feminine
+ и (Якщо -а) (Книжка Книжки)
+ і (Якщо -я) (Вулиця Вулиці)
### Neuter
+ а (Якщо -о) (Дерево Дерева)
+ я (Якщо -е) (Яйце Яйця)
+ і (Якщо -Я + double consonant) (Життя Житті)
+ Foreign borrowings don't change (Евро Евро)
