## Як
### Feminine singular
| Rule | Note |
|------|------|
| -а -> -о | мама -> мамо |
| -я -> -е | Столиця -> Столице |
| -я -> -ю | For names: Ілля -> Іллю |
| -ія -> -іє | Софія -> Софіє |

### Masculine singular
| Rule | Note |
|------|------|
| hard consonant -> -e | студент -> студенте |
| -г -> -ж | друг -> друже |
| -к -> -ч | "" |
| -х -> -ш | "" |
| family related -> -у | тато -> тату |
| -й -ь -> -ю | учитель -> учителю |
| -ко -ик -> -у | Марко -> Марку |
| -ович -> -у | Іван Петрович -> Іване Петровичу |
- чолові́к – чолові́че!
- хло́пець – хло́пче!
- друг – дру́же!
- Бог (‘God’) – Бо́же!
